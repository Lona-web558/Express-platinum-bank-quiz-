How the Code Vault works
  • Each question shows a real snippet from Platinum Bank with a blank in it.
  • Type the missing keyword, operator, or value and submit.
  • You have 3 security alerts (lives) — a wrong final answer or an expired timer costs one.
  • Correct answers build a streak for bonus points; hints cost 30 points but don't cost a life.
  • Clear every question, or run out of alerts, to reach the results screen.
